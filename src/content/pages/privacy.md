# 隐私政策 (Privacy Policy)

**最后更新日期：2026 年 2 月 1 日**

AIProxy Shop (https://aiproxy.shop) 尊重并保护您的个人隐私。本政策说明了我们如何处理您的个人信息。在使用我们的服务前，请您务必知晓我们收集数据的范围与用途。

---

## 1. 我们收集的信息

- **身份信息：** - 当您通过 **手机号** 注册或登录时，我们会收集您的手机号码及验证码。
  - 当您通过 **邮箱** 注册或登录时，我们会收集您的电子邮箱地址。
- **支付记录：** 我们记录您的充值流水及余额信息，但 **不直接存储** 您的银行卡号、支付密码或敏感支付凭据（支付过程由受监管的第三方支付网关处理）。
- **使用数据：** 为了计费、性能监控和排查故障，我们会记录 API 调用时间、调用的模型名称、消耗额度及调用者的 IP 地址。**我们不会记录或存储您 API 调用的具体内容（包括 Prompt 和回复结果）。**

## 2. 信息的使用方式

- **账户维护：** 用于管理您的账户余额、API 访问权限以及找回密码。
- **安全防范：** 监控异常流量和高频登录尝试，以防止恶意刷取额度、暴力破解及其他攻击行为。
- **服务通知：** 通过短信或邮件向您发送账户变动、服务更新或重要的安全警报。

## 3. 数据处理与隐私保护

- **内容隐私：** 我们 **不记录、不存储、不解析** 您通过 API 传输的具体对话内容。所有请求均通过流式或中转方式即时转发至底层模型供应商。
- **数据共享：** 我们不会将您的个人数据出售给第三方。您的请求数据将根据您调用的模型，加密转发至相应的底层供应商（如 OpenAI、Claude、Google Gemini 等）。

## 4. 您的权利

- **更正与更新：** 您可以在账户设置中修改您的登录密码或关联信息。
- **注销权：** 您可以随时联系我们注销账户。在身份核实无误后，我们将删除或匿名化处理您的所有关联身份数据。
- **数据安全：** 我们采取行业标准的加密措施保护您的手机号与邮箱信息不被泄露。

## 5. Cookie 的使用

我们使用 Cookie 保持您的登录状态及偏好设置。您可以根据浏览器设置禁用 Cookie，但这将导致您无法正常登录并使用 API 管理功能。
