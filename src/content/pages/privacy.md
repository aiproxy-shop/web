# 隐私政策 (Privacy Policy)

**最后更新日期：2026 年 1 月 10 日**

AIProxy Shop (https://aiproxy.shop) 尊重并保护您的个人隐私。本政策说明了我们如何处理您的个人信息。

## 1. 我们收集的信息

- **身份信息：** 当您通过 **邮箱、Google 或 GitHub** 登录时，我们会收集您的电子邮箱、用户 ID、昵称及头像。
- **支付记录：** 我们记录您的充值流水及余额信息，但**不直接存储**您的信用卡号或支付密码（支付过程由第三方支付网关处理）。
- **使用数据：** 为了计费和排查故障，我们会记录 API 调用时间、模型名称、消耗额度及调用者的 IP 地址。

## 2. 信息的使用方式

- **账户维护：** 用于管理您的账户余额和 API 访问权限。
- **防欺诈：** 监控异常流量，防止恶意刷取额度和攻击行为。
- **客户支持：** 回复您的技术咨询或通过邮件发送重要的服务通知。

## 3. 数据处理与隐私保护

- **对话内容：** 我们**不会主动存储**您的 API 请求的具体对话内容（Prompt 和 Completion），除非为了调试目的且经过您的同意，或法律法规另有要求。
- **数据共享：** 我们不会将您的个人数据出售给第三方。您的请求数据将根据您调用的模型转发至相应的供应商（如 OpenAI）。

## 4. 您的权利

- **注销权：** 您可以随时联系我们注销账户并删除所有关联的身份数据。
- **撤回授权：** 您可以随时在 Google 或 GitHub 设置中取消对 AIProxy Shop 的授权。

## 5. Cookie 的使用

我们使用 Cookie 保持您的登录状态及偏好设置。您可以根据浏览器设置禁用 Cookie，但可能导致无法登录。
