# 🚀 欢迎使用 AIProxy Shop

**让好用的 AI 模型触手可及。**

AIProxy Shop 是一个高性能的 AI 模型中转与分发平台。无需复杂的网络配置，通过一个统一的平台账户，您即可畅调全球最顶尖的语言模型（LLMs），享受极速、稳定且安全的服务体验。

---

## 核心特性

### 聚合接口，一键切换

**多厂商全场景支持。**
一个账户即可访问 **OpenAI, Claude, Gemini, Grok, DeepSeek, 智谱 AI** 等全球主流模型。无需维护多个平台的账号与支付方式，大幅降低集成成本，告别繁琐的跨境支付。

### 极速中转，国内直连

**100% 兼容原生标准。**
毫秒级延迟响应，支持国内环境直连。由于完全兼容业界主流的接口协议，您无需学习新框架，仅需修改基础接入地址（Base URL），即可实现现有业务的无缝迁移。

### 额度隔离，安全透明

**多 Key 管理与加密存储。**
支持为不同项目创建独立的 API Key，并为其分配专属额度上限。配合实时可视化账单看板，每一笔消耗都清晰可见。

> **安全声明**：所有密钥在数据库中均采用加密处理，且仅在创建时完整显示一次，确保您的资产安全。

## 安全与服务保障

- **数据隐私**：我们仅作为转发通道，严格遵守不保留用户请求明细的原则，保障您的数据隐私。
- **高可用性**：多渠道自动冗余切换，当上游某个供应商波动时，我们会尽力确保通道的连通性。
- **灵活配置**：支持自定义过期时间、单 Key 模型权限勾选，满足从个人开发者到企业团队的多场景需求。
